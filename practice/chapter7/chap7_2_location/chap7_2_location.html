表示ページのアドレス情報を取得／操作 - locationオブジェクト -
ボタンをクリックするなどの動作で、別ページに遷移したり、現在のページをリロードしたい
というような場合に利用するメソッド。

※「補足」欄にlocationオブジェクトで利用可能なプロパティ／メソッドを記載。

hash
概要 : アンカー名（#〜）
戻り値の例　: #gihyo?id=12345
http://www.wings.msn.to:8080/js/sample.html#gihyo?id=12345

host
概要 : ホスト（ホスト名+ポート番号。80の場合、ポート番号は省略。
戻り値の例　: wwww.wings.msn.to:8080
http://www.wings.msn.to:8080/js/sample.html#gihyo?id=12345

hostName
概要 : ホスト名
戻り値の例　: wwww.wings.msn.to
http://www.wings.msn.to:8080/js/sample.html#gihyo?id=12345

href
概要 : リンク先
戻り値の例　: http://www.wings.msn.to:8080/js/sample.html#gihyo?id=12345
http://www.wings.msn.to:8080/js/sample.html#gihyo?id=12345

pathname(読み取り専用)
概要 : パス名
戻り値の例　: js/sample.html
http://www.wings.msn.to:8080/js/sample.html#gihyo?id=12345

port(読み取り専用)
概要 : ポート番号
戻り値の例　: 8080
http://www.wings.msn.to:8080/js/sample.html#gihyo?id=12345

protocol(読み取り専用)
概要 : プロトコル
戻り値の例　: http:
http://www.wings.msn.to:8080/js/sample.html#gihyo?id=12345

search
概要 : クエリ情報
戻り値の例　: ?id=12345
http://www.wings.msn.to:8080/js/sample.html#gihyo?id=12345

reload()

履歴に沿ってページを前後に移動する - historyオブジェクト -
ブラウザのページ遷移の履歴を管理しているhistoryオブジェクトを利用すると、
ページの履歴を遡ったりまたは先に進むことができる。

back/forwardメソッド
history.backメソッド：履歴上の前ページへ遷移する。
history.forwardメソッド：履歴上の後ページへ遷移する。
